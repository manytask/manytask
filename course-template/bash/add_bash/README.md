# add (Bash)

Implement a shell function that echoes the sum of two integers.

## Signature

```bash
add() { ...; }   # echoes $1 + $2
```

Edit `add.sh` and implement `add`.

## Examples

```bash
add 2 3    # 5
add -1 1   # 0
add 0 0    # 0
```

## How to run tests locally

```bash
bash test_public.sh
```

The grader additionally runs a hidden `test_private.sh` with edge cases.
