# add (Go)

Implement a function that returns the sum of two integers.

## Signature

```go
func Add(a, b int) int
```

Edit `add.go` and implement `Add`.

## Examples

```go
Add(2, 3)   // 5
Add(-1, 1)  // 0
Add(0, 0)   // 0
```

## How to run tests locally

```bash
go run add.go test_public.go
```

The grader additionally runs a hidden `test_private.go` with edge cases.
